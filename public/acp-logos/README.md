# ACP provider logos

Drop official logos here so they appear in the chat ACP selector and Settings → ACP list. Use **SVG** for sharp scaling at any size.

## Expected filenames

| File            | Provider    |
|-----------------|-------------|
| `cursor.svg`    | Cursor      |
| `opencode.svg`  | OpenCode    |
| `codex.svg`     | Codex       |
| `claude-code.svg` | Claude Code |
| `gemini.svg`    | Gemini      |

## Where to get high-quality official logos

### Cursor
- **Official**: [cursor.com/brand](https://cursor.com/brand) — “Download brand assets” (ZIP with logos, app icons, avatars in 2D/2.5D, SVG/PNG). Use the **cube** or **wordmark** from the logo pack for the small selector; prefer SVG for crisp rendering.

### OpenCode
- **Official**: [opencode.ai/brand](https://opencode.ai/brand) — “Download all assets” for PNG and SVG. Use the SVG for best quality.

### Claude Code (Anthropic)
- **Official**: [Anthropic Newsroom on Brandfolder](https://brandfolder.com/anthropic/newsroom) — official Claude/Anthropic logos. Prefer SVG or high‑res PNG with transparent background.

### Codex
- If this is **GitHub Codex** or another product, use that project’s official brand or media kit (e.g. GitHub org, product website, or press page). Otherwise use your organization’s approved logo asset.

### Gemini
- **Official**: [Google AI branding](https://about.google/brand-resource-center/logos-list/) or [Gemini](https://gemini.google.com) — use the Gemini star/icon from Google’s brand resources or the product site. Prefer SVG for the selector.

---

After adding the files, the app will show them in the ACP dropdown and in Settings → ACP list. If a file is missing, the placeholder code icon is used.
